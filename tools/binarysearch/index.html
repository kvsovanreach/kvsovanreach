How It Works

Binary search works on sorted arrays by repeatedly dividing the search interval in half. It compares the target with the middle element:

  • If target equals mid, the element is found.
  • If target is less than mid, search the left half.
  • If target is greater than mid, search the right half.

This eliminates half the remaining elements each step, making it exponentially faster than linear search for large arrays.
