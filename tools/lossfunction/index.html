About Loss Functions

Loss functions measure the discrepancy between predicted and actual values. Choosing the right loss function affects model convergence, robustness, and sensitivity to outliers.

  • MSE: Penalizes large errors heavily due to squaring; standard for regression
  • MAE: Linear penalty, more robust to outliers than MSE
  • Huber: Combines MSE (small errors) and MAE (large errors) with a tunable delta
  • Cross-Entropy: Standard for classification; heavily penalizes confident wrong predictions
