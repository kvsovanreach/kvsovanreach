About Gradient Descent
1
Vanilla GD: Updates parameters using only the current gradient scaled by learning rate.
2
Momentum: Accumulates a velocity vector to accelerate through narrow ravines.
3
RMSProp: Adapts learning rate per-parameter using a moving average of squared gradients.
4
Adam: Combines momentum and RMSProp with bias correction for robust optimization.
