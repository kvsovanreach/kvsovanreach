Memory Concepts

Stack

LIFO structure for local variables. Fast allocation, automatic cleanup. Fixed size.

Heap

Dynamic memory for runtime allocation. Manual management required. Flexible size.

Fragmentation

Memory gaps from allocation/deallocation cycles. Reduces usable memory.

Memory Leak

Heap memory not freed. Accumulates over time. Causes performance issues.
