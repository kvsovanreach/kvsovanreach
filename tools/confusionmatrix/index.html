About Confusion Matrix Metrics

A confusion matrix summarizes the prediction results of a classifier. Each row represents actual classes, each column represents predicted classes.

  • Accuracy: Overall correct predictions / total predictions
  • Precision: True positives / (True positives + False positives)
  • Recall: True positives / (True positives + False negatives)
  • F1-Score: Harmonic mean of precision and recall
  • Specificity: True negatives / (True negatives + False positives)
  • MCC: Matthews Correlation Coefficient, balanced measure even for imbalanced data
