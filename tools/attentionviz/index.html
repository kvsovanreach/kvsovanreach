About Attention

Self-attention allows transformers to weigh the importance of different tokens when processing each position. The attention matrix shows how much each token "attends to" other tokens.

  • Query (Q): What information am I looking for?
  • Key (K): What information do I contain?
  • Value (V): What information do I provide?
