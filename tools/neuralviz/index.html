Sigmoid
Range (0, 1)
Vanishing Gradient Yes
Zero Centered No
Computationally Expensive

The sigmoid function squashes input values to a range between 0 and 1. It's commonly used in binary classification and in the output layer of neural networks.

Pros

  • Smooth gradient
  • Output bound between 0 and 1

Cons

  • Vanishing gradient problem
  • Not zero-centered
