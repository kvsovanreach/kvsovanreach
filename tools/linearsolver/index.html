About Linear Systems

A system of linear equations can have:

  • One unique solution - Lines intersect at one point
  • No solution - Parallel lines (inconsistent)
  • Infinite solutions - Same line (dependent)

Tip: Enter coefficients and constants. Leave empty for 0.
