About Quadratic Equations

A quadratic equation has the form ax2 + bx + c = 0 where a is not 0.

  • Two distinct real roots when discriminant > 0
  • One repeated real root when discriminant = 0
  • Two complex roots when discriminant < 0

Tip: The quadratic formula is x = (-b +/- sqrt(b2 - 4ac)) / (2a)
