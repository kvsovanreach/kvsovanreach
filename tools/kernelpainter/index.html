About Convolution

Convolution is a fundamental operation in image processing and neural networks. A small matrix (kernel) slides over the image, computing a weighted sum at each position.

Edge Detection: Finds boundaries between regions
Blur: Smooths the image by averaging neighbors
Sharpen: Enhances edges and details

Try it: Draw a simple shape, then switch between kernels to see how each filter transforms your image differently!
