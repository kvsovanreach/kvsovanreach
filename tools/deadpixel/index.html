Dead Pixel Test

How to Use

  1. Click "Start Test" to enter fullscreen mode
  2. Look carefully for any pixels that don't match the screen color
  3. Click or press arrow keys to cycle through colors
  4. Press Escape to exit fullscreen

Test Colors

Dead Pixels

Appear as black dots on any color. The pixel is permanently off.

Stuck Pixels

Appear as a single color (usually red, green, or blue) regardless of the screen color.
