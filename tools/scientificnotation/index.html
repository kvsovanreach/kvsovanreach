About Scientific Notation

Scientific notation expresses numbers as a x 10^n where 1 ≤ |a| < 10.

Engineering notation restricts the exponent to multiples of 3 (kilo, mega, giga, etc.).

  • Large numbers: 299,792,458 = 2.998 x 10^8
  • Small numbers: 0.00000001 = 1.0 x 10^-8
  • E-notation: 3.0e8 means 3.0 x 10^8

Tip: Use e-notation for input (e.g., 6.022e23) or type the full number.
