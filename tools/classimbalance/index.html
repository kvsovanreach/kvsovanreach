Understanding Class Imbalance

Class imbalance occurs when one class significantly outnumbers others. This causes models to be biased toward the majority class.

  • Oversampling: Duplicate minority class samples
  • Undersampling: Remove majority class samples
  • SMOTE: Generate synthetic minority samples
  • Class Weights: Penalize errors on minority class
