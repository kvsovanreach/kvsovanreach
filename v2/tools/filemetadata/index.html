About File Metadata

File metadata provides information about a file without accessing its content:

  • Name: The file's name including extension
  • Size: File size in bytes (converted to KB/MB/GB)
  • Type: MIME type identifying the file format
  • Modified: Last modification timestamp

Note: Files are processed locally. No data is uploaded to any server.
